
## Summary
In this chapter you learned how to create plots using Python and **matplotlib**. 
Types of charts:
 * line graphs
 * bar graphs
 * pie charts
 * bar and line graphs with error bars
 * scatter plots
 * histograms
 * box plots and violin plots
 * quiver plots
 * heat maps
