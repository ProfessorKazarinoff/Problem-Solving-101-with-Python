    Line Plots

    Line plots in matplotlib can be created using matplotlib's pyplot library First import matplotlib. If using a jupyter notebook include the line %matplotlib inline

    In [1]:
    import matplotlib.pyplot as plt
import numpy as np
%matplotlib inline

    In [4]:
    x = np.arange(0,4*np.pi,0.1)
y = np.sin(x)

    In [6]:
    plt.plot(x,y)
plt.show()
